# HC32L110C6PA 引脚定义

TSSOP20 封装，数据手册 Rev2.6。GPIO 逻辑编号 = Port×8 + Bit（P00=0, P01=1, ..., P14=12, ..., P36=30）。

---

| 功能 | 物理pin | GPIO逻辑编号 | 连接 | 备注 |

| VSS | 7 | — | GND | AVSS/DVSS |
| VCAP | 8 | — | 4.7μF 电容到 GND | LDO 内核供电 |
| NRST | 4 | P00=0 | 复位电路 | RESETB，低有效 |

## 3. PWM 输出

### 3.1 蜂鸣器 PWM

| 项目 | 值 |
|------|-----|
| 物理 pin | 1 |
| GPIO 名称 | **P34** |
| GPIO 逻辑编号 | **28** |
| 可用 Timer 功能 | TIM5_CHA, TIM4_CHA, TIM1_TOG, TIM0_EXT, PCA_CH0 |
| **分配 Timer** | **TIM4_CHA**（高级定时器，支持 PWM） |
| 连接 | 蜂鸣器驱动电路输入端 |
| 驱动电路类型 | 三极管放大，开漏输出 |
| 备注 | P25 也用了 TIM5_CHA，故蜂鸣器用 TIM4_CHA |

### 3.2 LED PWM

| 项目 | 值 |
|------|-----|
| 物理 pin | 15 |
| GPIO 名称 | **P25** |
| GPIO 逻辑编号 | **21** |
| 可用 Timer 功能 | TIM5_CHA, SPI_SCK, PCA_CH0, LPUART_RXD, I2C_SDA, TIM1_GATE |
| **分配 Timer** | **TIM5_CHA**（高级定时器，支持 PWM） |
| 连接 | LED 电源专用芯片输入端 |
| 驱动电路类型 | 芯片跟随 PWM 波形 |
| 备注 | P34 也用了 TIM5_CHA，故 LED 用 TIM5_CHA（两者错开） |

## 5. DIP 开关

### 5.1 声警报选择（2-bit, 公共端 GND, 低有效）

| 位 | 物理pin | GPIO名称 | GPIO逻辑编号 | 连接 | 备注 |
|----|--------|---------|-------------|------|------|
| BIT0 (LSB) | 19 | **P32** | **26** | DIP 开关 | UART1_TXD, TIM6_CHB, PCA_CH2 |
| BIT1 (MSB) | 20 | **P33** | **27** | DIP 开关 | UART1_RXD, TIM5_CHB, PCA_CH1 |

编码：
- 00：T3声音，小音（50% duty）
- 01：T3声音，大音（80% duty）
- 10：连续音，小音（50% duty）
- 11：连续音，大音（80% duty）

### 5.2 光警报选择（2-bit, 公共端 GND, 低有效）

| 位 | 物理pin | GPIO名称 | GPIO逻辑编号 | 连接 | 备注 |
|----|--------|---------|-------------|------|------|
| BIT0 (LSB) | 2 | **P35** | **29** | DIP 开关 | TIM6_CHB, TIM4_CHB, UART0_TXD |
| BIT1 (MSB) | 3 | **P36** | **30** | DIP 开关 | TIM6_CHA, TIM5_CHA, UART0_RXD |

编码：
- 00：低亮度（on 期间 30% duty）
- 01：中亮度（on 期间 60% duty）
- 10：高亮度（on 期间 100% duty）
- 11：预留，回退到低亮度
